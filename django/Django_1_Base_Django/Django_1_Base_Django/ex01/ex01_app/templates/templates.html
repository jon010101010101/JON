Django Template Engine

Blocks

Blocks allow child templates to override specific parts of parent templates.

{% verbatim %}{% block content %}{% endblock %}{% endverbatim %}

Loops

The for loop allows iteration over lists or dictionaries.

{% verbatim %}{% for item in items %}
    {{ item }}
{% endfor %}{% endverbatim %}

If Control Structures

If structures enable conditional execution of code in the template.

{% verbatim %}{% if condition %}
    Content if the condition is true
{% else %}
    Content if the condition is false
{% endif %}{% endverbatim %}

Context Variables

Context variables are displayed using double curly braces.

{% verbatim %}{{ variable_name }}{% endverbatim %}
