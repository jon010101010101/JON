Motor de Plantillas de Django

Bloques

Los bloques permiten que las plantillas hijas sobrescriban partes específicas de las plantillas padre.

{% verbatim %}{% block content %}{% endblock %}{% endverbatim %}

Bucles

Los bucles for permiten iterar sobre listas o diccionarios.

{% verbatim %}{% for item in items %}
    {{ item }}
{% endfor %}{% endverbatim %}

Estructuras de Control if

Las estructuras if permiten la ejecución condicional de código en la plantilla.

{% verbatim %}{% if condition %}
    Contenido si la condición es verdadera
{% else %}
    Contenido si la condición es falsa
{% endif %}{% endverbatim %}

Variables de Contexto

Las variables de contexto se muestran usando dobles llaves.

{% verbatim %}{{ variable_name }}{% endverbatim %}
