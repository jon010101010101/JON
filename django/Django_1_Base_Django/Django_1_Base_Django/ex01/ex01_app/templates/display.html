Proceso de Visualización de una Página Estática

  1. El navegador envía una solicitud HTTP a la URL especificada.
  2. Django recibe la solicitud y la pasa al URLconf para su procesamiento.
  3. El URLconf mapea la URL a la vista correspondiente.
  4. La vista procesa la solicitud y renderiza la plantilla.
  5. La plantilla se combina con cualquier dato contextual proporcionado por la vista.
  6. La vista devuelve una respuesta HTTP con el HTML generado.
  7. El navegador recibe la respuesta y muestra la página al usuario.
