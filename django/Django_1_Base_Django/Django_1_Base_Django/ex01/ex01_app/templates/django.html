Django: Framework Web

Django es un framework web de alto nivel escrito en Python que fomenta el desarrollo rápido y el diseño limpio y pragmático. Fue creado en 2003 por Adrian Holovaty y Simon Willison, y lanzado públicamente en 2005. Django sigue el patrón de diseño modelo-vista-template (MVT) y enfatiza la reutilización de código, el principio DRY (Don't Repeat Yourself) y la configuración automática.
