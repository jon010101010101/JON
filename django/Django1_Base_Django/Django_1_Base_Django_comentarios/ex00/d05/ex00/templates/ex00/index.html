Markdown Cheatsheet

Headers

# H1
## H2
### H3
#### H4
##### H5
###### H6

Emphasis

*italic* or _italic_
**bold** or __bold__
***bold italic*** or ___bold italic___

Lists

Unordered:
* Item 1
* Item 2
  * Subitem 2.1
  * Subitem 2.2

Ordered:
1. First item
2. Second item
3. Third item

Links

[Link text](URL)
[Link with title](URL "title")

Images

![Alt text](URL)
![Alt text](URL "Image title")

Blockquotes

> This is a blockquote
> It can span multiple lines

Code

Inline code: `code here`

Code block:
``` function example() {
  console.log("Hello, World!");
}
```

Horizontal Rule

---
or
***
or
___
